# Instructions Setup

## Step 1
- Install Docker from https://www.docker.com/products/docker-desktop/

## Step 2
- Clone this repositry: On your terminal git clone https://github.com/graciesearle/DESD.git

## Step 3
- Create a `.env` file and populate it accordingly with a DB password of your choosing and all relevant Secret Keys (*See `.env.example`*)

## Step 4
- To run (*IMPORTANT: THIS IS ONLY FOR Development*):
```bash
`bash setup.sh`
```

## Step 5
- You are now completely ready! Visit http://localhost:8000/marketplace for starters.
