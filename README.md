# DESD
Distributed Entriprise System Development Group Project.

To install Django:
1. Create venv:
    - On Windows: 
        ```
        python -m venv venv  
        ```
    - On Mac:
        ```
        python3 -m venv venv
        ```
    
2. Activate venv:
    - On Windows:
        ```
        venv\Scripts\activate
        ```
    - On Mac
        ```
        source venv/bin/activate
        ```
3. Run ``` pip install -r requirements.txt ```
